# pet-commerce-link-hub

## 專案目的

本專案為自建導流層（Link-in-bio），用於整合多品牌、多賣貨便賣場入口，
作為使用者進入賣場前的統一入口頁。

## 網頁連結

<https://bruce-yang-422.github.io/pet-commerce-link-hub/>

## 專案定位

- 僅作為「導流層 / 入口層」
- 不承載商品資訊
- 不顯示價格、SKU、規格
- 所有實際商品資料以賣貨便賣場為準

## 解決的問題

- 賣貨便賣場數量上限（每帳號最多 10 個）
- 單一賣場商品上限（100 商品）
- 商品分類能力不足
- 多品牌、多用品類導流管理困難

## 使用方式

使用者 → 本導流頁 → 對應賣貨便賣場

## 注意事項

任何商品描述、功效說明、規格資訊，請勿放入本專案。
